// Паралелограм
Паралелограм

Паралелограм з додатковою пружиною для підсилення тиску сошника на ґрунт — до 140 кг. Копіювальне колесо для точного налаштування глибини загортання насіння. Переналаштування міжряддя за допомогою одного болта.

// Однорядний сошник
Однорядний сошник — VELES-1

Висівний диск зі спеціальної високоміцної сталі. Дворядний підшипник — без обслуговування. Металевий захист диска від потрапляння ґрунту в міждисковий простір. Внутрішній чистик дисків. Армований насіннєпровід зі збільшеною міцністю.

// Дворядний сошник
Дворядний сошник — VELES-2

Два висівних диски на одному паралелограмі. Дворядковий посів з відстанню 10 см. Прикочувальні колеса зі спеціальним покриттям проти налипання землі. Чистик для запобігання налипанню землі на прикочувальні колеса.

// Висівний сошник

11 — Армований насіннєпровід зі збільшеною міцністю.
// Інокулянт
Опція інокулянт — LIXA-I

Система одночасного внесення інокулянту та/або рідких добрив безпосередньо в посівне ложе. Точне дозування через форсунку з отвором 0,6–1,0 мм під тиском — без світла та інших шкідливих для бактерій факторів.

// Система баків
Система баків — На рамі

Бак 600–2000 л безпосередньо на універсальній рамі. Компактно та стабільно. Для малих і середніх площ.

// Система баків
Система баків — Причіпний

Причіпний бак 2700–6000 л. Для великих площ та максимальної автономності. Автоматичне поповнення бака на рамі без зупинки.

// Технологія інокуляції
LIXA-I — Інокулянт прямо в посівне ложе

Система LIXA-I подає інокулянт безпосередньо в посівне ложе до насіння — через прецизійну форсунку з отвором 0,6–1,0 мм під тиском, без світла та інших шкідливих для бактерій факторів. Цей метод запобігає злипанню насіння у висівних органах та мінімізує ймовірність пропусків або подвійних висівів.

Переважно застосовується при посіві сої. Насіння потрапляє в посівне ложе разом зі стартовим добривом та рідким інокулянтом. Цей метод забезпечує:

  • Миттєве розчинення стартового добрива
  • Додаткове зволоження насіння — прискорює проростання на 2–3 дні
  • Мінімізацію загибелі бактерій порівняно з традиційною інокуляцією (протруюванням)

Інокульована культура:

  • Більш стійка на початковій фазі розвитку
  • Фіксує понад 50% необхідного азоту з повітря (близько 60 кг/га) та залишає його в ґрунті
  • Формує бульбочки на 10–12 день після сходів
