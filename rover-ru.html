// Модели & рабочие группы
5 моделей.
5 рабочих групп.

Каждая модель ROVER оптимизирована под определённый тип почвы и культуру. Все группы взаимозаменяемы и меняются менее чем за 30 минут.

ROVER-U
Стандартная обработка
Два пружинных зубца (виброзубья) с плоскими лапами. Предназначена для стандартной обработки на лёгких почвах. Эффективная борьба с сорняками при минимальном нарушении почвы.
Лёгкие почвы
ROVER-S
Усиленная обработка
Три пружинных зубца (виброзубья) с лапами стрельчатыми 150 мм. Для средних почв — усиленная обработка с повышенной проникающей способностью. Равномерное рыхление по всей рабочей ширине.
Средние почвы
ROVER-R
Интенсивное рыхление и окучивание
Модуль из 8 рыхлительных звёзд с угловой регулировкой в двух плоскостях — для окучивания или рыхления. С лапой стрельчатой 150 мм. Опционально: установка двух S-образных стоек (32×12) с лапами стрельчатыми 150 мм на каждый параллелограмм.
Окучивание
ROVER-P
Максимальное рыхление
Пять пружинных вибростоек с лапами стрельчатыми 150 мм. Для тяжёлых почв — максимальная глубина рыхления и проникновения. Интенсивная обработка даже на уплотнённых слоях почвы.
Тяжёлые почвы
ROVER-K
Комбинированная обработка
Две S-образные пружинные вибростойки и три пружинных зубца (виброзубья) с лапами стрельчатыми 150 мм. Комбинированная обработка — рыхление и финишная обработка за один проход.
Комбинированная
Для всех моделей доступны: лапы стрельчатые 200/250 мм и лапы 150 мм (стандарт).
// Компоненты и опции
Навесное оборудование &
системы защиты

Защитные диски, параллелограммы, гребневые бороны и защитные кожухи адаптируют ROVER точно под культуру и тип почвы. Все детали можно установить дополнительно.

// Система рулевого управления
Гидравлическая поперечная коррекция

Гидравлическая система рулевого управления смещает все рабочие группы синхронно поперёк направления движения — вручную или автоматически через EXACT. Диапазон поперечной регулировки ±25 см.

  • Диапазон регулировки ±25 см
  • Ручной или полностью автоматический режим
  • Время реакции менее 10 мс (с EXACT)
// Параллелограммы
Индивидуальное копирование рельефа для каждого ряда

Каждая рабочая группа подвешена на подпружиненном параллелограмме, который следует за профилем почвы — независимо от соседних рядов. Исключает заглубление на неровностях рельефа.

  • Подпружиненный 4-точечный параллелограмм
  • Компенсация ±6 см неровностей рельефа
  • Индивидуальная регулировка давления пружины
// Защитные диски
Защита растений непосредственно на рядке

Конические защитные диски на рабочем органе защищают молодое растение от засыпания землёй и механического повреждения. Регулируемое расстояние и угол в зависимости от стадии развития культуры.

  • Расстояние 5–30 мм до рядка
  • Для культур от фазы BBCH 11
  • Конические диски из закалённой стали
// Гребневые бороны
Борьба с сорняками в гребневой зоне

Гибкие пружинные бороны обрабатывают зону гребня между рядами — захватывают даже мельчайшие всходы сорняков в стадии нитевидного корня, которые не достают лапы культиватора.

  • Проволочная борона ∅ 4 мм, пружинная сталь
  • Скорость движения до 12 км/ч
  • Для картофеля, свёклы, кукурузы
// Защитные кожухи
Защита от засыпания для чувствительных культур

Откидные защитные кожухи рядом с рабочим органом предотвращают засыпание зоны растений землёй. Простая регулировка без инструмента — доступна с сиденья трактора.

  • Откидные, регулируются без инструмента
  • Для салата, шпината, моркови
  • Исполнение из нержавеющей стали
// Расширения
Совместимые
навесные системы

EXACT, LIXA, LIXA-O и GRANO можно устанавливать на любой ROVER по отдельности или в комбинации — для выполнения до 4 операций за один проход.

// Опция EXACT
Точность до 1 см — автоматически

Камерная система рядковедения

Оптическая система наведения EXACT распознаёт ряды в реальном времени и корректирует поперечную ось рабочих групп без GPS. Время реакции менее 10 мс, работа на скорости до 18 км/ч.

  • GPS не требуется
  • Работа 24/7, днём и ночью
  • Время реакции менее 10 мс

// Опция LIXA
Одновременное внесение жидких компонентов

Внесение жидких удобрений & средств защиты

LIXA вносит КАС, ЖКУ или средства защиты растений точно по рядам — одновременно с культивацией. До 3 жидких компонентов, бак 600–3000 л, управление по CAN-Bus.

  • До 3 компонентов одновременно
  • Бак 600–3000 л
  • Экономия гербицидов до −40%

// Опция LIXA-O
Листовая обработка над рядом растений

Внесение жидкости над рядом

LIXA-O дополняет систему LIXA форсунками над рядом — для листовых средств защиты растений, микроэлементов и регуляторов роста непосредственно на растение.

  • Форсунки над рядом, комбинируются с LIXA
  • Для листовых культур и овощей
  • Индивидуальное отключение по рядам через клапан

// Опция GRANO
Точное внесение гранул в прикорневую зону

Внесение гранулированных удобрений в прикорневую зону

GRANO вносит гранулированные удобрения непосредственно рядом или под ряд — во время культивации. Бак 800–2000 л, норма внесения 5–270 кг/га с бесступенчатой регулировкой через привод Accord.

  • Корпус из нержавеющей стали, минимальное обслуживание
  • Привод Accord — без электроники
  • Бак 800–2000 л
