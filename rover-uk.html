ROVER

// Моделі & робочі групи
5 моделей.
5 робочих груп.

Кожна модель ROVER оптимізована під певний тип ґрунту та культуру. Усі групи взаємозамінні та міняються менш ніж за 30 хвилин.

ROVER-U
Стандартний обробіток
Два пружинних зубці (віброзуби) з плоскими лапами. Призначена для стандартного обробітку на легких ґрунтах. Ефективна боротьба з бур'янами за мінімального порушення ґрунту.
Легкі ґрунти
ROVER-S
Посилений обробіток
Три пружинних зубці (віброзуби) з лапами стрілчастими 150 мм. Для середніх ґрунтів — посилений обробіток із підвищеною проникною здатністю. Рівномірне розпушування по всій робочій ширині.
Середні ґрунти
ROVER-R
Інтенсивне розпушування та загортання
Модуль із 8 розпушувальних зірок із кутовим регулюванням у двох площинах — для загортання або розпушування. З лапою стрілчастою 150 мм. Опціонально: встановлення двох S-подібних стійок (32×12) з лапами стрілчастими 150 мм на кожен паралелограм.
Загортання
ROVER-P
Максимальне розпушування
П'ять пружинних вібростійок із лапами стрілчастими 150 мм. Для важких ґрунтів — максимальна глибина розпушування та проникнення. Інтенсивний обробіток навіть на ущільнених шарах ґрунту.
Важкі ґрунти
ROVER-K
Комбінований обробіток
Дві S-подібні пружинні вібростійки та три пружинних зубці (віброзуби) з лапами стрілчастими 150 мм. Комбінований обробіток — розпушування та фінішна обробка за один прохід.
Комбінована
Для всіх моделей доступні: лапи стрілчасті 200/250 мм та лапи 150 мм (стандарт).
// Компоненти та опції
Навісне обладнання &
системи захисту

Захисні диски, паралелограми, гребеневі борони та захисні кожухи адаптують ROVER точно під культуру та тип ґрунту. Усі деталі можна встановити додатково.

// Система керування
Гідравлічна поперечна корекція

Гідравлічна система керування зміщує всі робочі групи синхронно поперек напрямку руху — вручну або автоматично через EXACT. Діапазон поперечного регулювання ±25 см.

  • Діапазон регулювання ±25 см
  • Ручний або повністю автоматичний режим
  • Час реакції менше 10 мс (з EXACT)
// Паралелограми
Індивідуальне копіювання рельєфу для кожного ряду

Кожна робоча група підвішена на підпружиненому паралелограмі, який слідує за профілем ґрунту — незалежно від сусідніх рядів. Виключає заглиблення на нерівностях рельєфу.

  • Підпружинений 4-точковий паралелограм
  • Компенсація ±6 см нерівностей рельєфу
  • Індивідуальне регулювання тиску пружини
// Захисні диски
Захист рослин безпосередньо на рядку

Конічні захисні диски на робочому органі захищають молоду рослину від засипання землею та механічного пошкодження. Регульована відстань та кут залежно від стадії розвитку культури.

  • Відстань 5–30 мм до рядка
  • Для культур від фази BBCH 11
  • Конічні диски з загартованої сталі
// Гребеневі борони
Боротьба з бур'янами у гребеневій зоні

Гнучкі пружинні борони обробляють зону гребеня між рядами — захоплюють навіть найдрібніші сходи бур'янів у стадії ниткоподібного кореня, яких не дістають лапи культиватора.

  • Дротова борона ∅ 4 мм, пружинна сталь
  • Швидкість руху до 12 км/год
  • Для картоплі, буряка, кукурудзи
// Захисні кожухи
Захист від засипання для чутливих культур

Відкидні захисні кожухи поряд із робочим органом запобігають засипанню зони рослин землею. Просте регулювання без інструменту — доступне з сидіння трактора.

  • Відкидні, регулюються без інструменту
  • Для салату, шпинату, моркви
  • Виконання з нержавіючої сталі
// Розширення
Сумісні
навісні системи

EXACT, LIXA, LIXA-O та GRANO можна встановлювати на будь-який ROVER окремо або в комбінації — для виконання до 4 операцій за один прохід.

// Опція EXACT
Точність до 1 см — автоматично

Камерна система рядковедення

Оптична система наведення EXACT розпізнає ряди в реальному часі та коригує поперечну вісь робочих груп без GPS. Час реакції менше 10 мс, робота на швидкості до 18 км/год.

  • GPS не потрібен
  • Робота 24/7, вдень і вночі
  • Час реакції менше 10 мс

// Опція LIXA
Одночасне внесення рідких компонентів

Внесення рідких добрив & засобів захисту

LIXA вносить КАС, РКД або засоби захисту рослин точно по рядах — одночасно з культивацією. До 3 рідких компонентів, бак 600–3000 л, керування по CAN-Bus.

  • До 3 компонентів одночасно
  • Бак 600–3000 л
  • Економія гербіцидів до −40%

// Опція LIXA-O
Листкова обробка над рядом рослин

Внесення рідини над рядом

LIXA-O доповнює систему LIXA форсунками над рядом — для листкових засобів захисту рослин, мікроелементів та регуляторів росту безпосередньо на рослину.

  • Форсунки над рядом, комбінуються з LIXA
  • Для листкових культур та овочів
  • Індивідуальне вимкнення по рядах через клапан

// Опція GRANO
Точне внесення гранул у прикореневу зону

Внесення гранульованих добрив у прикореневу зону

GRANO вносить гранульовані добрива безпосередньо поруч або під ряд — під час культивації. Бак 800–2000 л, норма внесення 5–270 кг/га з безступінчастим регулюванням через привід Accord.

  • Корпус із нержавіючої сталі, мінімальне обслуговування
  • Привід Accord — без електроніки
  • Бак 800–2000 л
