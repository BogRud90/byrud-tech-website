// Инокулянт
Опция инокулянт — LIXA-I

Система одновременного внесения инокулянта и/или жидких удобрений непосредственно в посевное ложе. Точная дозировка через форсунку с отверстием 0,6–1,0 мм под давлением — без света и других вредных для бактерий факторов.

// Система баков
Система баков — На раме

Бак 600–2000 л непосредственно на универсальной раме. Компактно и надёжно. Для малых и средних площадей.

// Прицепной бак
Система баков — Прицепной

Прицепной бак 2700–6000 л. Для больших площадей и максимальной автономности. Автоматическое пополнение бака на раме без остановки.

// Технология инокуляции
LIXA-I — Инокулянт прямо в посевное ложе

Система LIXA-I подаёт инокулянт непосредственно в посевное ложе к семенам — через прецизионную форсунку с отверстием 0,6–1,0 мм под давлением, без света и других вредных для бактерий факторов. Этот метод предотвращает склеивание семян в высевающих органах и минимизирует вероятность пропусков или двойных высевов.

Применяется преимущественно при посеве сои. Семена поступают в посевное ложе со стартовым удобрением и жидким инокулянтом. Этот метод обеспечивает:

  • Мгновенное растворение стартового удобрения
  • Дополнительное увлажнение семян — ускоряет прорастание на 2–3 дня
  • Минимизацию гибели бактерий по сравнению с традиционной инокуляцией (протравливанием)

Инокулированная культура:

  • Более устойчива на начальной стадии развития
  • Фиксирует более 50% необходимого азота из воздуха (ок. 60 кг/га) и оставляет его в почве
  • Формирует клубеньки на 10–12 день после всходов
