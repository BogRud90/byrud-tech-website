// Ротаційна борона
STERN

Ротаційна борона для цілорічного використання — підготовка посівного ложа, весняний обробіток озимих культур, розбивання кірки після посіву буряків, внесення гранульованих або рідких добрив із загортанням у ґрунт, підсів трав, суцільний висів культур під сидерати. Ротаційна борона STERN розроблена для цілорічної експлуатації:

  • Підготовка посівного ложа
  • Весняний обробіток озимих культур
  • Розбивання кірки після посіву (напр. буряки)
  • Внесення гранульованих або рідких добрив із загортанням
  • Підсів трав
  • Суцільний висів культур під сидерати

// Зірчасті колеса
Зірчасті колеса — 40–90 кг тиску пружини, 1 болт на ряд

Литі зірчасті колеса з тиском пружини 40–90 кг на ряд — рівномірний обробіток по всій ширині. Регулювання одним болтом на ряд — швидка і проста адаптація до різних глибин обробітку. Надійна конструкція, мінімальне обслуговування.

// Опція добрива та насіння
Одночасне боронування, удобрення та посів

Опціональний бункер для добрив та насіння для одночасного внесення — боронування, удобрення та посів за один прохід. Ідеально для прямого посіву та вирощування проміжних культур.

  • Бункер для добрив та насіння комбінований
  • Ідеально для посіву проміжних культур
  • Менше проходів, більше ефективності
