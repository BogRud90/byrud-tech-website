Ротаційна борона STERN розроблена для цілорічної експлуатації:

// Зірчасті колеса
Зірчасті колеса — 50 кг тиску пружини, 1 болт на ряд

Литі зірчасті колеса з тиском пружини 50 кг на ряд — рівномірний обробіток по всій ширині. Регулювання одним болтом на ряд — швидка і проста адаптація до різних глибин обробітку. Надійна конструкція, мінімальне обслуговування.

// Опція добрива та насіння
Одночасне боронування, удобрення та посів

Опціональний бункер для добрив та насіння для одночасного внесення — боронування, удобрення та посів за один прохід. Ідеально для прямого посіву та вирощування проміжних культур.

  • Бункер для добрив та насіння комбінований
  • Ідеально для посіву проміжних культур
  • Менше проходів, більше ефективності
